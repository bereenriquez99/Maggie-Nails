# Maggie Nails

Sitio web para Maggie Nails, un salón de manicura y pedicura premium con una experiencia natural, femenina y relajante.

## Descripción

Este proyecto es un sitio web estático que presenta los servicios de Maggie Nails, incluyendo paquetes básicos, spa y premium, galería de imágenes, reseñas y contacto.

## Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone <url-del-repositorio>
   cd maggie-nails
   ```

2. Abre `index.html` en tu navegador web.

No se requieren dependencias adicionales, ya que es un sitio estático.

## Tecnologías Utilizadas

- **HTML5**: Estructura del sitio
- **CSS3**: Estilos y diseño responsivo
- **JavaScript**: Interactividad básica (menú móvil, animaciones)
